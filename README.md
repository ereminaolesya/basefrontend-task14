## Описание

Реализация HTML+CSS+JavaScript проект автокомплит поля с динамическим поиском и фильтрацией списка городов.
Задача взята из учебного практического курса.

## Требования к задаче

Источник данных:
Массив городов: статический список городов для поиска
Case-insensitive search: поиск без учета регистра
Partial match: поиск только по частичному совпадению - по началу слова

Поведение поиска:
Real-time filtering: фильтрация при каждом изменении input
Минимальная длина: поиск начинается c 1-го символа
Пустой input: скрытие списка при очистке поля
Максимум результатов: без ограничений количества показываемых вариантов
Выбор элемента: значение input становится равно выбранной опции, и выпадающий список скрывается

Обработка событий с клавиатуры:
Если текстовое поле предоставляет поисковую выдачу и фокус находится на текстовом поле, то
У пользователя появляется возможность навигироваться по предоставленным опциям с использованием клавиш-стрелок на клавиатуре
Изначально фокуса нет ни на одной опции. Инициировать выбор можно только нажатием на стрелку “вниз”
При изменении поисковой выдачи фокус на опции должен пропадать (если он есть)
Навигирование должно быть “зацикленным” - если пользователь находится на самом нижнем элементе и нажимает на стрелочку “вниз”, фокус должен смещаться на самую верхнюю опцию
При нажатии на “Enter” происходит выбор опции, на которой был фокус, и значение текстового поля должно быть равно выбранной опции. Поисковая выдача исчезает
Элемент, на котором происходит фокус, должен содержать css-класс active

Семантика тегов:
Лист опций и сами опции должны соответствовать семантике списка и его элементов
Текстовое поле и его label
